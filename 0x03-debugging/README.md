# 0x03. C - Debugging

## Learning Objectives
- Understand what debugging is and why it matters
- Learn manual debugging techniques
- Read error messages effectively
- Fix code with logical errors

## Project Structure
- `main.h` - Header with all function prototypes
- `0-main.c` - Test file for positive_or_negative with input 0
- `1-main.c` - Comment out infinite loop code
- `2-largest_number.c` - Fix function to find largest number
- `3-print_remaining_days.c` - Fix leap year calculation

## Debugging Principles
1. Read the error message carefully
2. Trace through the code line by line
3. Use printf statements to see values
4. Check edge cases
5. Verify assumptions
