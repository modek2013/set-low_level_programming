# 0x01. C - Variables, if, else, while

## Learning Objectives
- Understanding arithmetic operators, logical operators, and relational operators
- Using if, if...else statements for decision making
- Using while loops for iteration
- Working with variables and data types
- Understanding ASCII character set

## Tasks
0. Positive anything is better than negative nothing - if/else conditional
1. The last digit - modulo operator and conditionals
2. Alphabet game - while loop with putchar
3. alphABET - lowercase and uppercase with putchar
4. Alphabet soup - skipping letters with conditionals
5. Numbers - printing digits with putchar
6. Numberz - printing digits without char variables
7. Smile in the mirror - reverse alphabet
8. Hexadecimal - base16 printing
9. Combinations - comma-separated digits

## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style checker
