## Learning Objectives
- Understanding the C compilation process
- Using gcc with specific flags
- Following Betty coding style
- Creating shell scripts for compilation tasks

## Tasks
0. Preprocessor - Script to run C file through preprocessor
1. Compiler - Script to compile but not link
2. Assembler - Script to generate assembly code
3. Name - Script to compile with custom output name
4. Hello, puts - C program using puts
5. Hello, printf - C program using printf
6. Size - C program to print sizes of types

## Environment
- Ubuntu 20.04 LTS
- gcc with flags: -Wall -Werror -Wextra -pedantic -std=gnu89
- Betty style checker
